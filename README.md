# ⚡ Watt Vision : AI

> *Unmasking the invisible heartbeat of your home.*

Watt Vision : AI is a Machine Learning + Electrical Engineering project that analyzes household energy data and predicts which appliances are active using electrical signatures from the mains power supply.

Built using **Python**, **Streamlit**, and **Scikit-learn**.

---

## 🚀 Features

- Real-time mains power visualization
- AI-based appliance detection
- RMS Current & Power Factor analysis
- Dynamic live charts
- Futuristic Streamlit interface

### Detectable Appliances
- Fridge
- Kettle
- TV
- Microwave
- Washing Machine

---

## 🧠 Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Joblib

---

## ⚙️ How It Works

The system reads:

- Mains power
- Current
- Voltage

Then generates electrical features such as:

- Power Change
- Rolling Mean
- Rolling Standard Deviation
- RMS Current
- Power Factor

Machine Learning models then predict whether each appliance is **ON** or **OFF**.
